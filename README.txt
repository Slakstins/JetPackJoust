This game was developed by Seth Lakstins and Kyle Brown as a final project for the class Object Oriented Software Development(CSSE220) at RHIT. 

Features include: 
	20 levels
	functional custom animations
	Scorekeeping and a combo meter
	A final boss
	7 different mobs
	
For some images and details, view Presentation.pptx in this directory.

To launch the game, clone the project and run Main.java

Controls:
	Moving: WASD or arrow keys
	Pausing: P
	Next level: U

If you have any questions or feedback, email lakstise@rose-hulman.edu

Thanks for playing :)
